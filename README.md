# STEP Viewer (OpenCascade + Three.js)

Een volledig client-side STEP/ STP viewer gebouwd met:

- OpenCascade WebAssembly (`occt-import-js`)
- Three.js (voor 3D weergave)
- Zero backend – **geen uploads naar servers**

Dit project verwerkt STEP-bestanden uitsluitend **lokaal in de browser**.

---

## ✨ Features
- Upload elk `.step` of `.stp` bestand
- Voorbeeldbestand (demo STEP)
- Refresh knop voor een nieuwe sessie
- OpenCascade converteert STEP → mesh in WebAssembly
- Three.js: orbit, zoom en pan
- Volledig offline te gebruiken (na éénmalig laden)

Voorbeeldpagina online hier: https://pimo131.github.io/STEPviewer/
